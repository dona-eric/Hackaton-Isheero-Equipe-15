# Hackaton-Equipe-15
Transformer des données mondiales en connaissance locale. La base GDELT surveille en permanence les médias du monde entier dans plus de 100 langues. Notre mission : en extraire tout ce qui concerne le Bénin des 12 derniers mois, l'analyser, et en faire des insights utiles pour un journaliste, un chercheur, un décideur public.
